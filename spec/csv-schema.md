# uaf_payload.csv Schema（UAF v1.0）

## 1. 文件身份

| 属性 | 值 |
|------|-----|
| 文件名 | `uaf_payload.csv` |
| 编码 | UTF-8，**无 BOM** |
| 行数 | 2 行（1 行表头 + 1 行数据） |
| 列数 | 4 列（固定顺序） |

## 2. 表头

第一行必须为（无空格）：

```csv
subject,date,content,tags
```

## 3. 字段定义

### 3.1 subject

- **类型**：字符串
- **说明**：作业科目，如 `数学`、`语文`
- **约束**：非空；长度 ≤ 200 字符

### 3.2 date

- **类型**：字符串（ISO 8601）
- **说明**：布置时间
- **推荐格式**：
  - `YYYY-MM-DD`（如 `2026-05-19`）
  - 或完整日期时间 `YYYY-MM-DDTHH:mm:ss`（可带时区）
- **约束**：必须可被 ISO 8601 解析器解析

### 3.3 content

- **类型**：字符串
- **说明**：作业正文，支持换行
- **约束**：非空；长度 ≤ 2000 字符
- **转义**：含逗号、双引号或换行时，整字段用双引号包裹，内部 `"` 写为 `""`（RFC 4180）

### 3.4 tags

- **类型**：字符串（逻辑层为字符串数组）
- **说明**：作业标签
- **CSV 表示**：
  - 无标签：空字符串 ``
  - 有标签：英文分号 `;` 分隔，如 `必做;几何;重难点`
- **约束**：单个标签非空；标签内不得包含 `;`；标签总数 ≤ 20；单标签长度 ≤ 50

## 4. RFC 4180 规则摘要

1. 字段以逗号分隔
2. 含特殊字符的字段用 `"` 包裹
3. 字段内 `"` 转义为 `""`
4. 换行符可出现在引号字段内

## 5. 示例

### 5.1 简单作业

```csv
subject,date,content,tags
数学,2026-05-19,完成课本第45页第1、2题，请拍照上传。,必做;几何
```

### 5.2 多行正文

```csv
subject,date,content,tags
语文,2026-05-19,"背诵《静夜思》
第二段抄写生字",必做
```

### 5.3 无标签

```csv
subject,date,content,tags
英语,2026-05-19,朗读课文 Unit 3,
```

## 6. 逻辑层映射

解析后映射为：

```typescript
interface UafPayload {
  subject: string;
  date: string;
  content: string;
  tags: string[];  // 由 tags 列 split(';')，过滤空串
}
```

序列化时 `tags.join(';')` 写回 CSV 列。
