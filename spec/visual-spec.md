# UAF 作业卡片视觉规范 v1.0

## 1. 页面

| 属性 | 值 |
|------|-----|
| 纸张 | A4 |
| 方向 | 纵向（Portrait） |
| 页数 | **1**（禁止分页） |
| 尺寸 | 595.28 × 841.89 pt（ISO A4） |
| 页面背景 | `#F8FAFC`（浅灰，可选） |

## 2. 布局（顶对齐，非整页铺满）

对标班级作业看板（如 Classworks）：**卡片自页面顶部起排布**，左右留白，**高度随内容收缩**，禁止用边框占满整张 A4。

| 区域 | 说明 |
|------|------|
| 页边距 | 40 pt（四边） |
| 卡片宽度 | 页面宽度 − 80 pt |
| 卡片位置 | **顶对齐**：卡片上缘距页顶 40 pt |
| 卡片高度 | 由内边距 + 顶栏 + 正文 + 标签区计算，**不得**固定为整页高 |
| 卡片圆角 | 16 pt |
| 卡片背景 | `#FFFFFF`，1 pt 描边 `#E2E8F0` |
| 浅阴影 | 可选：偏移 2 pt 的浅灰圆角底层（矢量，非位图） |

坐标系：PDF 默认左下角为原点，Y 向上。

## 3. 顶栏（卡片内）

### 3.1 科目胶囊

- 背景色：`#2563EB`
- 文字色：`#FFFFFF`
- 字号：14 pt
- 内边距：水平 16 pt，垂直 8 pt
- 形状：**胶囊**（圆角 = 高度的一半）

### 3.2 日期胶囊

- 背景色：`#F1F5F9`
- 文字色：`#334155`
- 字号：12 pt
- 内边距：水平 12 pt，垂直 6 pt
- 形状：胶囊
- 位置：顶栏**右侧**，与科目胶囊垂直居中对齐

### 3.3 日期显示格式

将 ISO `date` 格式化为 `YYYY年M月D日`（展示层；CSV 仍为 ISO）。

## 4. 正文区

- 区内边距：24 pt（相对卡片边缘）
- 字号：**22 pt**（默认）
- 字色：`#0F172A`
- 行高：1.5 × 字号
- 对齐：左对齐
- 字体：嵌入中文无衬线体（参考实现：Noto Sans SC **子集**）

### 4.1 自适应

超出卡片内可用高度时，按 22 → 18 → 16 → 14 pt 阶梯缩小；14 pt 仍不足则末尾 `…` 截断。

## 5. 底栏标签

- 每个 tag 为独立 **chip**（胶囊形）
- 背景色：`#E0E7FF`，文字色：`#3730A3`
- 字号：11 pt；内边距：水平 10 pt，垂直 5 pt
- chip 间距：8 pt；最多 2 行
- 位于正文下方，与正文间距 20 pt

## 6. 文件体积

- 展示层为 **矢量文字**，禁止将卡片 rasterize 为位图嵌入。
- 字体须 **子集嵌入**（仅包含本次作业用到的字形）。
- 典型单文件：**< 500 KB**（含内嵌 `uaf_payload.csv`）。

## 7. PDF 导出水印

为标识该 PDF 由 UAF 标准导出，须在页面右下角添加**固定水印文本**。水印同时也是防伪与溯源的辅助手段。

### 7.1 水印文本

固定内容（大小写不敏感，推荐全大写）：

```
使用 UAF 导出
```

允许带版本号扩展：

```
使用 UAF v1.0 导出
```

### 7.2 水印位置

| 属性 | 值 |
|------|-----|
| 水平位置 | 右对齐，距页面右边缘 **40 pt** |
| 垂直位置 | 底部，距页面下边缘 **40 pt** |
| 基准点 | 文本**右下角**对齐（即 x、y 坐标为文字右下角的基准点） |

### 7.3 水印样式

| 属性 | 值 |
|------|-----|
| 字号 | **10 pt** |
| 字色 | `#94A3B8`（浅灰） |
| 字体 | 与正文一致即可（参考实现使用 Noto Sans SC 子集） |
| 透明度 | 50%（若 PDF 引擎支持），否则以浅灰色模拟 |
| 背景/描边 | 无 |
| 旋转角度 | 0°（水平放置） |

### 7.4 视觉行为

- 水印应**始终可见**，不随卡片内容自适应而被遮挡。
- 当卡片阴影或卡片本体延伸至页面右下角附近时，水印可叠加于卡片上方，凭借低透明度与浅色实现视觉分层。

## 8. 无障碍与兼容

- 正文与背景对比度 ≥ 4.5:1
- 水印与页面背景对比度 ≤ 2:1，避免干扰阅读
- 展示层自包含，普通 PDF 阅读器可直接阅读
